# AREN Athletic Static Export

This is a deployable static export of the AREN Athletic site.

## Included
- index.html
- route folders with index.html files
- /static CSS, JS, and image assets
- privacy and terms pages

## Routes
- /
- /about
- /training
- /services
- /street-miles
- /shop
- /contact
- /privacy
- /terms

## Deploy
You can deploy this folder directly to:
- Vercel
- Cloudflare Pages
- Netlify

Build command: none
Output directory: .

## Local preview
Use any static server, for example:

npx serve .
